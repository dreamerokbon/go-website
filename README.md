## 🧠 專案簡介（Overview）
本網站以「圍棋」為主題，介紹中、日、韓、台四大棋院與圍棋文化。  
本專案旨在練習 **HTML 結構化設計與 CSS 版面配置**，透過多頁式靜態網站的形式，  
加強對網站導覽、排版、表格與圖片呈現的整合能力。

---

## ⚙️ 使用技術（Tech Stack）
- **HTML5**：結構化內容、語意化標籤應用  
- **CSS3 / Flexbox**：頁面排版與響應式配置  
- **Google Charts**：在日本棋院頁面繪製圓餅圖  
- **Canvas-nest.js**：背景粒子特效  
- **GitHub Pages**：靜態網站部署  

---

## 🌟 功能特色（Features）
- 多頁式導覽：分別介紹中、日、韓、台棋院  
- 響應式版面：以 Flexbox 調整版面結構  
- 使用表格展示比賽與棋士資訊  
- 使用圖片與圖表輔助閱讀  
- 全站統一導覽列與風格設計
